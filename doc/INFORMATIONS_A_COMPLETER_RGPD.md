## Informations générales

### Notes importantes

1. **SIRET et RCS** : Ces informations sont obligatoires pour les entreprises en France. Elles peuvent être trouvées sur :
   - Le K-bis de l'entreprise
   - Le site infogreffe.fr
   - Le site societe.com

2. **Numéro TVA** : Obligatoire uniquement si l'entreprise est assujettie à la TVA.

3. **Médiateur** : Obligatoire pour les entreprises de consommation. Vous pouvez adhérer à un médiateur de la consommation.

4. **Hébergeur** : Les informations de l'hébergeur sont généralement disponibles dans les conditions générales d'utilisation de l'hébergeur.

5. **Directeur de publication** : Personne responsable du contenu du site. Généralement le dirigeant de l'entreprise ou le responsable communication.
